âš–ï¸ Class Balancing Techniques Implemented

Class imbalance was handled using the following four techniques:

Random Oversampling

Increases minority class samples by duplicating existing data points.

Random Undersampling

Reduces majority class samples to balance the dataset.

SMOTE (Synthetic Minority Over-sampling Technique)

Generates synthetic data points for the minority class.

Class Weight Adjustment

Assigns higher weight to minority class during model training.

Learning:
Balancing techniques significantly improve model fairness and performance on imbalanced datasets.
